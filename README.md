# Supabase manager - Spring Boot template

This is a template for a spring boot, specifically for a Supabase database. Right now, it just contains the simplest 
configuration, which already supabase contains, however is just a skeleton for other projects.

## Purpose

Template already configured, for a quick start in other projects

## Free of use

If you are interseted in learning spring boot, in here there are several basic concepts like Controllers, centralize 
error handling and database access. Or just a quick start for your projects.

## Warning

In this template, the application.properties is missing, as well, contains secrets, like the nuclear launching codes.
So, have in mind that you ming need to add it
